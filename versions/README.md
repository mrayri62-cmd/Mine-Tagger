# Version Pack

This repository root tracks the latest `1.21.11` source.

The `versions` folders contain per-version jars and matching source snapshots for the full `1.21` to `1.21.11` range.

Shared source lines:
- `1.21` and `1.21.1` use the `1.21` source line.
- `1.21.2` and `1.21.3` use the `1.21.2` source line.
- `1.21.6`, `1.21.7`, and `1.21.8` use the `1.21.6` source line.
- `1.21.9` and `1.21.10` use the `1.21.9` source line.
