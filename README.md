# Tier Tagger
A gorgeous minecraft fabric mod which displays everyone's tier from the Vanilla Pvp Tierlist in game like this: "Ht1 | Ooh_Netiyiy"

Very smol but very useful mod; you'll likey likey :)

Modrinth: https://modrinth.com/project/plR1TgE7

## Repository layout

- `main` tracks the latest `1.21.11` source line.
- Older Minecraft versions are available as dedicated Git branches such as `1.21`, `1.21.1`, `1.21.2`, and so on.
- Some nearby versions share the same code line when upstream support was identical.
